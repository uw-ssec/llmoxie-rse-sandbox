# Archived: climate-model packaging track

This directory holds the retired **"Package a Climate Model with the Research
Loop"** track, archived on 2026-06-12. It is no longer wired into the sandbox
walkthrough or the README — the active guided demo is the ocean research arc
(`docs/slides/research-loop-ocean.md` with `samples/ocean/`).

Contents, preserved at their original repository paths:

- `docs/slides/research-loop.md` — the guided Marp deck that walked
  `/researching → /planning-implementations → /implementing-plans →
  /validating-implementations` to package the climate scripts.
- `samples/climate_model.py` — the toy seasonal temperature model.
- `samples/climate_data_analysis.py` — analysis over `climate_data.csv`.
- `samples/climate_data.csv` — the sample data.
- `samples/model_visualization.py` — the plotting script.

The deck references the `sandbox` Marp theme, which still lives at
`docs/slides/themes/`. To restore the track, move these files back to their
original paths and re-add the walkthrough button and README references
(see git history for this directory).
