# ⏳ Your sandbox is getting ready

First launch takes a few minutes while the devcontainer prepares everything:

1. the **pixi** Python environment
2. the **LLMoxie Model Provider** extension (routes Copilot Chat through the
   LLMoxie / LiteLLM gateway)
3. the **Copilot CLI** with the RSE Agent plugins
4. the **Get Started walkthrough**

You can watch progress in the terminal panel. When the setup output finishes,
reload the window — `Ctrl+Shift+P` → **Developer: Reload Window** — and the
**LLMoxie RSE Sandbox walkthrough** will open and take it from there.

It is safe to close this tab at any time. To find the walkthrough later:
**Help → Welcome → LLMoxie RSE Sandbox**.
